테두리 스타일

border-style : 테두리 스타일 지정

선택자{border-[방향]-style : none | hidden | dashed | dotted | double | groove | inset | outset | ridge | solid}

border-width : 테두리 두께 지정

선택자{border-[방향]-width : thin | medium | thick | px}

border-color : 테두리 색상 지정

선택자{border-[방향]-color : 색상값}

border : 테두리 스타일 한번에 설정

선택자{border-[방향] : width style color}

border-radius : 박스 모서리를 둥글게 하는 속성

선택자{border-[방향]-radius : 숫자}

box-shadow : box에 그림자를 주는 속성

선택자{box-shadow : 수평거리 수직거리 흐림정도 번짐정도 색상}

여백스타일

margin : 요소와 요소 사이의 거리

        선택자 {margin-[위치] : 거리;} -> top,bottom,left,right 선택해서 거리 설정
        선택자 {margin : 거리;} -> 4방향 모두 한번에 설정
        선택자 {margin : 거리 거리;} -> 거리1: top,bottom, 거리2: left,right

padding : 테두리(border)와 컨텐츠 사이의 거리

선택자 {padding-[위치] : 거리;}
