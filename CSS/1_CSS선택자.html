선택자
HTML문서 내부에서 스타일을 적용하고 싶은 요소(HTML태그)를 선택할 때 사용하는 기능
요소를 선택해서 원하는 스타일과 기능을 적용

전체 선택자

HTML 문서 안의 모든 태그를 선택할 때 사용하는 선택자

* {속성 : 값;} 형식으로 사용

태그 선택자

HTML 문서 안의 동일한 태그를 모두 선택하는 선택자

태그이름{속성 : 값;}

아이디 선택자

HTML 문서 내에 해당하는 id속성을 가진 태그만 선택할 때 사용(id는 중복x) 

#아이디{속성 : 값;} 형식으로 사용

클래스 선택자

HTML 문서 내 여러개 태그를 동일한 클래스 이름으로 지정하여 선택할 때 사용 

.클래스이름{속성 : 값;}
