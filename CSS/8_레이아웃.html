포지셔닝 스타일

box-sizing : width/height의 범위를 설정하는 속성

선택자{box-sizing : content-box | border-box}

position : 페이지 요소들의 위치를 설정하는 속성

선택자{position : static | relative | absolute | fixed}















visibility : 페이지에 요소를 보이거나 보이지않게 하는 속성

선택자{visibility : visible | hidden}

opacity : 요소의 투명도를 설정

선택자{opacity : 0 ~ 1}

z-index : 요소들을 쌓는 순서를 지정

선택자{z-index: 순서숫자}












float : 요소의 배치를 왼쪽 또는 오른쪽으로 지정하는 속성

선택자{float : left | right | none}

clear : float으로 인해 다음 요소가 받는 영향을 초기화하기위한 속성

선택자{clear : left | right | both | none}
